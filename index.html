Projects

Letter Classification

Constructed training model to recognize physical letters from the Latin alphabet through the process of persistent homology. Utilized Lower Star Image Filtration from Ripser package in python to locate the optimized life and death pair for each single letter in zero-dimensional homology. Successfully trained the computer to recognize Latin alphabet letters through exact algorithm with in optimal time.


Marketing Research

Marketing Research is a project to determine the probability that a customer will buy a “Kid Creative” magazine. The data given had a predictor variable with values 0 or 1 so a logistic regression was preformed. With R, we obtained the MLE estimates for the coefficients of the logistic model as well as the odds ratios. Then, after running a backwards selection, we used a deviance test to compare the full model to the simplified model. The final outcome produced an equation for “Kid Creative” magazine that predicts the probability that it will be brought.


Boston Housing Regression

Boston Housing Regression is a model that predicts the median value of an owner-occupied home. To produce this model, a multiple linear regression using all 12 regressors was run in R. By using modeling concepts such as interaction terms, transformation of variables, polynomial terms, and various stepwise methods, I successfully developed a model to predict the median value of an owner-occupied home.
